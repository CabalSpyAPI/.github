# CabalSpy — KOL Wallet Tracker & Smart Money API

**Real-time multi-chain data layer for labeled KOL, Smart Money and Whale wallets.**

CabalSpy is a KOL wallet tracker and Smart Money wallet tracker covering Solana, Ethereum, BNB Chain, Base and Robinhood. 2000+ tracked wallets, 30+ REST endpoints and 6 WebSocket streams deliver real-time KOL data and Smart Money data the moment a tracked wallet trades — no polling required.

## Use cases

- **Wallet Tracking** — Full trader profile: stats, realized PnL, live holdings and a PnL calendar for any tracked wallet. *(Solana, ETH, BNB, Base, Robinhood)*
- **Leaderboard** — Rank KOL, Smart Money and Whale wallets by PnL, win rate, trades or volume. *(Solana, ETH, BNB, Base, Robinhood)*
- **Wallet Count** — Live badge showing how many tracked KOL wallets are in a token right now. *(Solana, ETH, BNB, Base, Robinhood)*
- **Chart Events** — Live buy/sell bubbles on a price chart the moment a KOL trades. *(Solana, ETH, BNB, Base, Robinhood)*
- **Supply Holding** — Real-time % of token supply held by tracked wallets. *(Solana, ETH, BNB, Base, Robinhood)*
- **Signal Service** — Cluster detection and live entry/exit alerts when multiple KOL or Smart Money wallets buy together. *(Solana, ETH, BNB, Base, Robinhood)*
- **Real-Time Holding Table** — Live holder table merging position, native balance and bundle flags. *(Solana only)*
- **Widget** — Embeddable live wallet bubble map, one iframe. *(Solana, BNB, Base, Robinhood)*
- **KOL Bundle Detection** — Detect coordinated KOL + side-wallet buys with a confidence score. *(Solana only)*
- **PnL Calendar** — Day-by-day realized PnL calendar, win/loss streaks at a glance. *(Solana, ETH, BNB, Base, Robinhood)*

## What we offer

- **KOL API / Smart Money API** — 30+ REST endpoints for leaderboards, wallet stats, transaction feeds, signals and per-token analytics.
- **WebSocket (Realtime KOL & Smart Money Data)** — 6 live streams: transactions, holders, balances, bundles, signals and wallet counts.
- **Widget** — Embeddable KOL wallet tracker widget for any token or wallet address.
- **Dashboard** — Web-based leaderboard at [cabalspy.xyz](https://cabalspy.xyz/) with PNL rankings, win rates and wallet visualizations.
- **AlphaBot** — AI call bot with a 72% hit rate, surfacing high-signal token calls from real-time KOL activity.
- **Chrome Extension** — Scan any token and see which KOLs/Smart Money wallets are in it, how much, and their win rate.
